# LoopDownloader
## What is this?
これは、URLに連番が含まれているファイルを順番に自動でダウンロードできるPythonスクリプトです。

## Usage
`python LoopDownloader.py {連番部分を「num」に置き換えたurl} {ファイルの拡張子}`

## Uses modules
- requests  
`pip install requests`
- PIL  
`pip install pillow`
